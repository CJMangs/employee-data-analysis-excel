## 📊 Employee Data Analysis Dashboard (Excel)

### 📌 Project Overview

This project demonstrates the use of Microsoft Excel for data cleaning, transformation, and visualization.
An interactive dashboard was created to analyze employee salary distribution, department performance, and workforce insights.

---

### 🔧 Key Features

* Data cleaning using TRIM
* Text extraction (First Name and Last Name)
* Automated email generation
* Text transformation using UPPER, LOWER, and PROPER functions
* Aggregation using:

  * SUM
  * AVERAGE
  * COUNTA
  * SUMIF / SUMIFS
  * AVERAGEIF / AVERAGEIFS
  * COUNTIFS
  * MAXIFS

---

### 📊 Dashboard Insights

* Total Salary
* Average Salary
* Total Employees
* Total Salary by Department
* Average Salary by Department
* Employee Count by Department
* Total Salary by City
